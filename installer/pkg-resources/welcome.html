Latigid LP Generator

This installer sets up the LP Generator MCP server on your Mac. It will:

  • Install Node.js if not already present (via Homebrew or official installer)
  • Download the latest version from GitHub
  • Configure Claude Desktop to use it
  • Restart Claude Desktop so the changes take effect

After clicking Install, a Terminal window will open to ask for:

  • Your Anthropic API key (personal — get one from console.anthropic.com)
  • The HubSpot Client Secret (sent to you on Slack)
  • The Auth Secret (sent to you on Slack)
Have those three values ready before you continue.
The installer will ask for them once Terminal opens.
